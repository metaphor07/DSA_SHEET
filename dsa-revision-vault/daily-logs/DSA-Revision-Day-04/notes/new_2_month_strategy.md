# Updated 2-Month Revision Strategy

## Change from Day 5 onward
Do not do one topic for many days continuously.

Use a rotating high-frequency topic cycle:
1. Arrays / Hashing / Prefix
2. Sliding Window / Two Pointers
3. Stack / Monotonic Stack
4. Binary Search
5. Trees
6. Graphs
7. Heap / Greedy / Intervals
8. Dynamic Programming
9. Backtracking / Recursion
10. Mixed mock / contest-style

## Daily rule
Each day:
- 2 questions from today's main topic
- 1 question from yesterday's weak topic
- 1 question from an older weak topic
- 1 mixed unseen medium

This gives spaced repetition and mixed recall.
