# Dynamic Programming

Use this folder for problems involving:

- 1D DP
- 2D DP
- Knapsack
- LIS
- DP on strings
- Grid DP

## Common triggers

- "Count ways" → DP
- "Maximum/minimum cost" → DP
- "Longest subsequence/subarray" → DP or sliding window depending on continuity
- "Choose/not choose" → Knapsack style DP
