# Graphs

Use this folder for problems involving:

- BFS/DFS traversal
- Connected components
- Topological sort
- Cycle detection
- Dijkstra
- DSU
- Grid traversal

## Common triggers

- "Number of islands/components" → DFS/BFS/DSU
- "Prerequisites/order" → Topological sort
- "Shortest path unweighted" → BFS
- "Shortest path weighted positive" → Dijkstra
