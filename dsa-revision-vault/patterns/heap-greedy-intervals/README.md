# Heap, Greedy & Intervals

Use this folder for problems involving:

- Top K
- K-way merge
- Meeting rooms
- Merge intervals
- Scheduling
- Greedy choice

## Common triggers

- "Top K" → Heap
- "Smallest/largest repeatedly" → Heap
- "Overlapping intervals" → Sort + merge
- "Minimum rooms/platforms" → Min heap or sweep line
